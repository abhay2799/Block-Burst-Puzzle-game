# Block Blast!

A fully interactive Block Burst puzzle game built with Phaser 4 and procedural sound effects.

## How to Play

1. Drag and drop blocks onto the 8x8 grid
2. Complete full rows or columns to clear them and earn points
3. Place all 3 pieces to get a new set
4. Game ends when no piece can fit on the board

## Features

- 39 unique block shapes (no rotation)
- Satisfying line-clear animations with particles and screen shake
- Procedural sound effects (ZzFX) - no audio files needed
- Combo multiplier for consecutive-turn clears
- High score saved to localStorage
- Responsive design (mobile + desktop)
- Beautiful color palette and visual feedback

## Tech Stack

- **Phaser 4** - Game framework
- **ZzFX** - Procedural sound synthesis
- **Vite** - Build tool

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
